1.选择器
	1.标签选择器
		p{
		}
	2.id选择
		id不可重复
		#id{
		}
	3.类选择器
		.div{
		}
	4.交集选择器
		既要是a标签又要是class为div 的 标签
			a.div{
			}
	5.后台选择器
		查找标签的后代
		#id p{
		}
	6.子代选择器
		只找标签的儿子
		#id > p{
		}

2.三种样式
	行内样式：写在某个标签中，只对本行有效。（内联样式）
	内嵌样式：也在某个页面中，只对本页面有效。前面的例子都是内嵌样式。
		<style type="text/css">
		</style>
	外部样式：写在某个单独的.CSS文件中，对引用该文件的页面有效。
		<link rel="stylesheet" type="text/css" href="css/cssOfC.css"/>
		
3.样式优先级
	1.行内样式>ID选择器的优先级>CLASS选择器>标签选择器
	2.同级别里，样式只认最后书写
		比如说 对同一个id选择写了两次，系统只认最后一次
