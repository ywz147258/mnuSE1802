1.统计学生的平均分，平均分大于60分就是及格，小于60分就是不及格，取前三名
	先统计学生的平均分
	case when 来输出 及格还是不及格
	再用limit取前三名
	
	SELECT
	student_id,
	score as '分数',
	case when avg( score ) <60
	then '不及格'
	else '及格'
	end as '评价'
	FROM
		grade 
	GROUP BY
		student_id
	limit 3
	
2.统计出每一个同学，每一个科目的平均分，最高分和最低分
	A 语文 50
	A 英语 60
	A 语文 120
	
	学生	科目	平均分
	A 		语文 	60
	A 		英语 	60
	
	SELECT
	student_id,
	course_id,
	count(1) as '考试次数',
	avg( score ) AS '平均分',
	max( score ) AS '最高分',
	min( score ) AS '最低分' 
	FROM
		grade 
	GROUP BY
		student_id,
		course_id;
		
3.查询课程1比课程2成绩高的所有学生的学号.
SELECT
	g1.student_id,
	g1.course_id,
	g2.course_id,
	g1.score,
	g2.score
FROM
	grade g1,
	grade g2 
WHERE
	g1.student_id = g2.student_id 
	AND g1.score > g2.score 
	AND g1.course_id = 1 
	AND g2.course_id = 2 
GROUP BY
	g1.student_id

4.对学生的成绩进统计，筛选出同一个科目下，
最高分比最低分高出60分的同学，并输出 学号、科目、最高分、最低分
SELECT
	student_id,
	course_id,
	max( score ) AS '最高分',
	min( score ) AS '最低分' 
	FROM
		grade 
	GROUP BY
		student_id,
		course_id
	having (max(score)-min(score))>10;
